# Average Birthday

Client-side calculator for finding the average birthday between several people on a yearly circle.

## How it works

- Add names and birth dates.
- Each date is converted to an angle on the year circle.
- The app sums the date vectors and shows the resulting average date.
- The birth year is used so leap years are handled correctly.

## Usage

Open `index.html` in a browser or publish the repository with GitHub Pages.

Live page: https://davidmz.github.io/average-birthday/
